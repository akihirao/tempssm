Practice I: Analysis of Monthly Air Temperature

Data set

The ThermoSSM package provides an example dataset with csv format. It contains a monthly temperature time series measured at Mount Akadake, Hokkaido, Japan. The data originated from: https://www.biodic.go.jp/moni1000/findings/data/index.html. Contents of the csv file as follows:

Year, Month, Temp

2010,8,13.6

2010,9,6.8

2010,10,0.2

…

Supplementary code and test data for estimating sea temperature trends using a linear Gaussian state-space model.

loading dataset and converting ts object

baba_data <- read_csv("https://raw.githubusercontent.com/logics-of-blue/sea-temperature-trend-jogashima/refs/heads/main/src/ssm-sample-data.csv")

# time　　　：日付(論文では0.5か月だが、本サンプルコードでは月単位データとした)
# temp　　　：水温
# distance　：黒潮流軸までの距離
# kuroshio_a：黒潮流路がA型のときに1をとるフラグ
head(baba_data)
## # A tibble: 6 × 4
##   time        temp distance kuroshio_a

## 4 1998-04-01  16.5      131          0
## 5 1998-05-01  19.7       45          0

# 秋冬の黒潮Aフラグの追加
baba_data <- 
  baba_data %>% 
  mutate(winter = as.numeric(month(as.Date(time)) %in% c(11, 12, 1, 2, 3)),
         winter_a = winter * kuroshio_a)

# winter_a：秋冬かつ、黒潮がAであるときに1をとるフラグ

Utility function of jma_sst2ts()

This function downloads publicly available daily mean sea-surface temperature (SST) data for Japanese coastal waters

provided by the Japan Meteorological Agency (JMA),and returns the monthly average data as a object.
